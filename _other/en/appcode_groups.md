**Groups** is feature from Appcode(XCode). Plugin doesn't support **Groups**, it creates common dirs.
At the moment you can create **Group** manually (from existing files\directories).
To do this, select **Files** perspective in the Project View to make created files visible, then add them via context menu (RMC on the directory).

P.S. If template contains files only (FileTemplates), then files will be added to **Groups** successfully.
