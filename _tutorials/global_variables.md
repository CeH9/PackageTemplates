Глобальные переменные это переменные которые доступны в:

* [File Templates][1]
* Directory name
* File name

Использование:<br>**Example<font color="#9400D3">${VARIABLE_NAME}</font>File**

Если VARIABLE_NAME = FooBar тогда

результат: имя = ExampleFooBarFile
